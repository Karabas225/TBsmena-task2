# T-Библиотека

T-Библиотека - это консольная программа для управления личной библиотекой. Все данные сохраняются в JSON-файл.

## Возможности

- Добавление книги (название, автор, жанр, год, описание).
- Просмотр всех книг с фильтрацией по жанру и статусу прочтения, сортировкой.
- Изменение статуса прочтения.
- Добавление/удаление из избранного.
- Просмотр избранных книг.
- Удаление книги по ID.
- Поиск по ключевым словам.
- Статистика: общее количество, прочитанные, непрочитанные, избранные, любимый жанр (по избранным).
- Импорт книг из другой библиотеки (JSON-файл) с пропуском дубликатов (по названию+автору). Открывается диалог выбора файла.
- Корректировка данных книги по ID.

## Технологии

- Python 3
- Библиотека Rich для красивого консольного интерфейса
- Хранение данных в JSON

## Структура проекта

```
t-library/
|---- src/                          # Исходный код
|   |---- models.py                 # Класс Book — как устроена книга
|   |---- storage.py                # Загрузка и сохранение JSON
|   |---- library.py                # Основная логика (добавление, удаление, поиск)
|   |---- ui.py                     # Всё, что связано с интерфейсом (меню, таблицы)
|   |---- utils.py                  # Вспомогательные функции (очистка экрана и т.п.)
|---- data/                          # Папка для файла с книгами
|   |---- library.json               # Файл, куда сохраняются книги
|---- requirements.txt                # Список необходимых библиотек
|---- README.md                       # Описание проекта
|---- READMEofficial.md
|---- main.py                         # Точка входа: запуск программы
```
Программа проста в использовании: все действия выбираются из меню цифрами, а на каждом шаге есть подсказки. Разобраться сможет любой, кто хоть раз работал в командной строке.
