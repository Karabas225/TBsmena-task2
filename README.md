# T-Библиотека

T-Библиотека - это консольная программа для тех, кто хочет держать свою домашнюю библиотеку в порядке. Она помогает записывать книги, следить за тем, что уже прочитано, отмечать любимое и быстро находить нужное. Все данные сохраняются в файл, так что после закрытия программы ничего не теряется.

## Что умеет программа

- **Добавлять книги.** При добавлении указывается название, автор, жанр, год выпуска и краткое описание. Каждая книга получает свой номер (ID).
- **Показывать все книги.** Список выводится в виде таблицы. Можно отфильтровать книги по жанру или по тому, прочитаны они или нет, а также отсортировать по названию, автору или году.
- **Отмечать прочитанное.** Книгу можно пометить как «прочитана» или «не прочитана».
- **Работать с избранным.** Понравившиеся книги добавляются в избранное (отмечаются звёздочкой). Потом можно посмотреть только избранные книги или убрать их оттуда.
- **Искать книги.** Поиск работает по словам из названия, имени автора или описания.
- **Удалять книги.** Если книга больше не нужна, её можно удалить по ID.
- **Сохранять данные.** После каждого изменения библиотека автоматически сохраняется в файл `data/library.json`. При следующем запуске все книги снова загружаются.
- **Предотвращать ошибки.** Программа не даст дважды добавить одну и ту же книгу в избранное или сменить статус прочтения на тот же самый.

## Технологии

- **Язык:** Python 3.
- **Для красивого вывода используется библиотека Rich** (версия 13.7.0) - она рисует таблицы, панели и добавляет цвета.
- **Данные хранятся в формате JSON.**

## Структура проекта

```
t-library/
|---- src/                          # Исходный код
|   |---- models.py                 # Класс Book — как устроена книга
|   |---- storage.py                # Загрузка и сохранение JSON
|   |---- library.py                # Основная логика (добавление, удаление, поиск)
|   |---- ui.py                     # Всё, что связано с интерфейсом (меню, таблицы)
|   |---- utils.py                  # Вспомогательные функции (очистка экрана и т.п.)
|---- data/                          # Папка для файла с книгами
|   |---- library.json               # Файл, куда сохраняются книги
|---- requirements.txt                # Список необходимых библиотек
|---- README.md                       # Описание проекта
|---- READMEofficial.md
|---- main.py                         # Точка входа: запуск программы
```

Программа проста в использовании: все действия выбираются из меню цифрами, а на каждом шаге есть подсказки. Разобраться сможет любой, кто хоть раз работал в командной строке.
